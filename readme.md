# 坚果云API

坚果云的 API，感谢[蓝奏云API](https://github.com/zaxtyson/LanZouCloud-API)提供的参考。

## 简介

封装了坚果云的基础功能：

+ 获取账户基本信息
+ 文件、文件夹及回收站内容的增删改查
+ 文件与操作历史、应用密码、书签及分享列表的基本操作

项目还在完善中。。
